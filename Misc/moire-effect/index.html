Moiré Effect

Two near-identical periodic patterns stacked together create large, slow fringes — the same idea as audio beats, but in space. Choose a pattern, then nudge layer B's spacing or rotation and watch the Moiré pitch explode. The predicted fringe spacing is shown live: for a small rotation θ between equal grids, dM = d / (2 sin(θ/2)); for parallel grids of slightly different spacings, dM = d1d2 / |d1−d2|.
